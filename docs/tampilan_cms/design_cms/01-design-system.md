# Tasklist: Design System CMS

Fokus pada standarisasi elemen visual agar dashboard konsisten, profesional, dan mudah digunakan.

## 1. 🎨 Fondasi Visual
- [ ] **Define:** Palet Warna Admin.
  - [ ] Warna Primer (Brand), Sekunder, Neutral (Grays), dan Semantic (Success, Danger, Warning).
- [ ] **Define:** Tipografi Dashboard.
  - [ ] Skala font yang clean untuk keterbacaan data tabel dan form yang padat.

## 2. 🧩 Komponen UI Dasar (Atomic)
- [ ] **Design:** Buttons & Links.
  - [ ] **Create:** Variasi Primary, Secondary, Outline, Ghost, dan Destructive (Danger).
- [ ] **Design:** Form Inputs.
  - [ ] **Create:** Style untuk Text Input, Textarea, Select/Dropdown, Checkbox, dan Toggle Switch.
  - [ ] **Create:** State untuk Hover, Focus, Disabled, dan Error.
- [ ] **Design:** Data Display.
  - [ ] **Create:** Badge/Label status (Published, Draft, Upcoming, dll).
  - [ ] **Create:** Avatar placeholder untuk user/penulis.

## 3. 🏢 Komponen Kompleks (Molecules/Organisms)
- [ ] **Design:** Data Tables.
  - [ ] **Create:** Layout baris tabel, header sortable, dan komponen paginasi.
- [ ] **Design:** Modals & Dialogs.
  - [ ] **Create:** UI Modal konfirmasi hapus dan pop-up form ringan.
- [ ] **Design:** WYSIWYG Editor Toolbar.
  - [ ] **Create:** Desain kustom untuk toolbar teks editor agar menyatu dengan style dashboard.
- [ ] **Design:** Feedback UI.
  - [ ] **Create:** Desain Toast Notification dan Loading Skeleton.
